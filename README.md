# 📦 Sistema de Cestas Básicas Personalizadas

## 🤝 Sobre a Sociedade São Vicente de Paulo

A Sociedade São Vicente de Paulo (SSVP) é uma organização católica de caridade que realiza ações sociais, como visitas a famílias em situação de vulnerabilidade e distribuição de cestas básicas.

## 🧠 Sobre o projeto

Este projeto tem como objetivo melhorar a forma como cestas básicas são distribuídas na SSVP.

Ao invés de receber uma cesta padronizada, o assistido poderá escolher os itens que realmente precisa, utilizando um saldo mensal fictício disponibilizado pelo sistema.

---

## 🎯 Objetivo

Permitir que assistidos montem suas próprias cestas básicas de forma simples, garantindo uma distribuição mais eficiente e personalizada dos alimentos.

---

## ⚙️ Como funciona

- O assistido recebe um saldo mensal
- Escolhe os produtos disponíveis no sistema
- Monta sua cesta de acordo com sua necessidade
- Finaliza o pedido dentro do limite de saldo

O vicentino (responsável pela instituição) poderá:

- Visualizar os pedidos realizados
- Separar as cestas
- Atualizar o status dos pedidos
